HEXO+Github搭建个人博客

发表于 2019-06-06 | 更新于 2019-06-12 | 分类于 hexo

配置环境

安装Node
作用：用来生成静态页面的
到node官网下载安装即可，可执行node -v检查node版本
安装Git
作用：把本地的项目提交到github上去.
到git官网下载安装即可,作为一个前端开发人员这两个想必一定都早已经安装好了
申请github账号
作用：最为我们代码的远程仓库，托管我们的项目
怎么注册GitHub账号也不多说了，GitHub对于前端开发人员非常重要，很多优秀开放人员都在git上发布了自己的项目，是我们学习和研究的一个必备途径

安装hexo

Hexo 是一个快速、简洁且高效的博客框架。Hexo 使用 Markdown（或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页。
在安装之前先创建一个空文件夹（blog）,存放hexo配置文件,我们之后的操作都在这个文件夹下进行。

启动本地服务

一切还没有结束，因为这只是我们本地启了一个服务而已，接下来还有其他工作需要做。看下面一步步操作

配置Github创建一个仓库

登陆github,点击右上角的加号，点击New repository,Repository name为your_user_name.github.io这个是固定的，在勾选上Initialize this repository with a README点击Create repository按钮即可。可以直接访问your_user_name.github.io，这就是以后我们存放静态网页的网址。

本地关联git

修改本地的配置文件_config.yml，来建立关联，修改如下：repo地址是使用SSH协议
deploy:
type: git
repo: git@github.com:luyanhua/luyanhua.github.io.git
branch: master
然后执行

本地工程目录执行git init，每次更改后部署都要执行hexo generate， hexo deploy就可以在我们的网址访问。
