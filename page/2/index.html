HEXO主题修改

发表于 2019-05-28 | 更新于 2019-06-10 | 分类于 hexo

一默认主题

我们通过 hexo init 初始化的网站，使用的是 hexo 默认的主题样式为 lanscape，位于 xxx/themes/lanscape 目录，我们可以到官网的主题 根据自己的喜好更换其它的主题。

1.下载新主题（next）

将 Next 主题下载到本地 themes 目录下的 next 文件夹中与lanscape平级。

2.修改配置文件_config.yml

注意是next文件夹下面的配置文件
找到Schemes模块，修稿scheme:Pisces大家可以根据自己的喜好修改，可以自己测试是各种主题的样式。

3.自定义主题菜单

我使用的主题是next,比较简洁的一种，菜单就只有四个首页 标签 分类 归档，当我们初始化之后就只有两个标签怎么在新添加两个呢？另外还有个问题所有的文字是英文的我们怎么修改成中文的呢？
主题菜单是在主题的配置文件 blog/themes/next/_config.yml 中 menu 模块定义的如下：
menu:
home: / || home

#about: /about/ || user
tags: /tags/ || tags
categories: /categories/ || th 这里是对应的图标
archives: /archives/ || archive

#schedule: /schedule/ || calendar

#sitemap: /sitemap.xml || sitemap

#commonweal: /404/ || heartbeat

Enable / Disable menu icons / item badges.

menu_settings:
icons: true 这个icons必须设置成true
badges: false
自己搭建的时候要多尝试可以试着解开注释看效果
其中使用的菜单图标都是来自于 Font Awesome ，可以在这个网站找到自己喜欢的图标后，把图标名称写在 || 后面即可。

4.404报错（添加分类的内容）

此时，当我们在首页点击 标签 或 分类 时，会报 404 ，这是因为我们还没有创建对应的文件夹，所以报 404 找不到。
创建标签和分类对应的文件夹

执行成功以后，本地的 source 目录下生成了对应的文件夹，且文件夹中都会有一个 index.md ：
在index.md中添加type属性分别是type: tags type:categories
添加文章在source下面的_posts下面增加一个md文件，向下面这样指定categories: hexo和tags: hexo

title: HEXO+Github搭建个人博客
categories: hexo
tags: hexo

那么怎么修改语言呢？

修改语言只需要修改最外层的配置文件，修改language。
language: zh-CN
